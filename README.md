🚀 Project Launch: L'Ambroisie | High-End Fine-Dining Web App 🍽️

Most restaurant websites rely on flat, static PDF menus. I wanted to build something different—a seamless web experience that merges luxury editorial aesthetics with a snappy, reactive frontend projects

⚡ Technical Highlights
Dynamic Cart Logic: Engineered a real-time ordering engine. The application handles on-the-fly calculations for totals, item quantities, and updates states seamlessly depending on the user's fulfillment choice (Delivery, Pickup, Pre-order).

Responsive Layout Architecture: Built a completely fluid, mobile-first interface. Leveraging CSS Grid for asymmetric layouts and Flexbox for component alignment, the site scales flawlessly from ultra-wide monitors down to mobile screens.

Semantic Code Structure: Wrote clean, valid HTML5 to ensure the site is highly accessible, lightweight, and structured properly for SEO.

Performance Optimization: Balanced high-quality, luxury food imagery with fast page-load speeds by optimizing asset pipelines and ensuring interactive states (hovers, active clicks) remain butter-smooth.

🧠 Lessons Learned
State Synchronization: Handling real-time arithmetic updates between individual menu cards and a global basket component without framework overhead was a fantastic exercise in pure, logical frontend design.

Micro-interactions matter: I realized that subtle animations and responsive hover effects are what elevate a website from looking "good" to feeling premium.

💻 Tech Stack Used
Frontend: Semantic HTML5, Advanced CSS3 (Grid & Flexbox),  JavaScript 

Deployment: GitHub Pages
